The main difference between local storage and session storage is the data's lifespan.
There are also some difference between them.
They are:
1. Local storage across browser sessions where Session storage Only for the current browser session.
2. Local storage Storage capacity 10 MB on the other hand Session storage capacity is only 5 MB.
3. Local storage Storing data that the user needs to access even when they are not actively using the website but session storage Storing data that is only needed for a short period of time.

In JavaScript, scope determines the visibility and accessibility of variables and functions within different parts of your code.
There are two main types of scope: global scope and block scope.
1.Global scope exists until the program terminates but Block scope exists until the end of the block in which they are declared
2. Global scope declaras Outside of any function or block of code but Block scope declears inside a block of code.
3. Global scope is Accessible from anywhere in the program on the other hand Block scope Only accessible from within the block in which they are declared.

The JavaScript event loop is a single-threaded loop that is responsible for executing JavaScript code and handling events.It is a fundamental part of the JavaScript runtime environment that allows it to handle asynchronous operations efficiently without blocking the main execution thread. It works by maintaining two queues: the call stack and the event queue.
